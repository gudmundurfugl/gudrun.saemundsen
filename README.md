# Nafn systur — Kynningarsíða

Þetta er einföld, statísk síða fyrir GitHub Pages.

## Fljótleg notkun

1. **Endurnefndu** `index.html` með réttu nafni og settu inn raunverulegar upplýsingar (nafn, netfang, tengla).
2. **Uppfærðu greinalista** í `index.html` (sjá &lt;ul&gt; → `Pistlar og greinar`). Settu inn rétta slóð ef til er.
3. **Push-aðu** skrám í public repo (`main` branch).
4. Kveiktu á **Settings → Pages** og veldu `Deploy from a branch` → `main` → `/ (root)`.

## Lén (valfrjálst)

- Bættu við `CNAME` skrá í rót repo með textanum `dæmi.is` ef þú notar sérlén.
- Stilltu svo A/ALIAS eða CNAME í DNS á `username.github.io`.

## Sérsníða

- Breyttu litum í `styles.css` (`:root` breytur).
- Skiptu út avatar fyrir eigin mynd (t.d. `avatar.jpg`) og settu í rót repo.
